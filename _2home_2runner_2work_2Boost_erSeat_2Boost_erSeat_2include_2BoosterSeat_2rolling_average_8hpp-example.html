This class declares a rolling average. This class is used to calculate a rolling average over a set interval. The two things that can be changed are the window size and the rate at which the user calls the intervalCall() method.

Window size = 60 minutes intervalCall() is called every minute This will give you the average per minute over the last 60 minutes.
