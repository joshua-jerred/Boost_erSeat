Detailed Description

template<class StopwatchClock = clck::Clock>
class bst::Stopwatch< StopwatchClock >

A simple stopwatch class. Basic start (resume), stop (pause), and reset functionality.

Template Parameters
StopwatchClock- The clock to use for timing. Defaults to the high_resolution_clock.
